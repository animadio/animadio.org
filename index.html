Animadio

CSS Framework

STABLE VERSION WOULD BE AVAILABLE THE 1st OF JUNE !!

How to prepare for takeoff ?

Immediate takeoff with Animadio, the only free & open-source framework that offers real interactivity to the click & much more :

  • Eight types of selectors working with hover, focus, active & checked states
  • Hundreds of class combinations by concatenation
  • Flexbox & Grid systems included with common options
  • Click anywhere on the page to enable & disable any design effect anywhere else
  • Management of keyframes animations & their options
  • Complete & simple customization with directly modifiable CSS variables
  • More than 99% of pure CSS ; so all of that without JavaScript, it's a joke ?... No ?!

CSS Only

Animadio is a CSS framework whose library is composed only of CSS source code: jQuery, or even Javascript, are not included or needed in production...

Grunt is the only JavaScript app used in the framework developement side for recursive tasks (concat, source map, webkit & min) to build the library files (but it can be used to concat CSS elements of your choice by forking the repository)

You can simply use the library & if you want or if you need : you can overload the CSS variables.

Browsers

Animadio uses PostCSS with autoprefixer to provide a maximum of compatibility for all main browsers, including mainly Firefox, Chrome & Opera, secondarily Edge, Safari & Explorer (for some effects, especially all Skew animations)

Installation

  • CDN

    During development, add this tag to your HTML :

    <link href="https://cdn.jsdelivr.net/npm/animadio@0.3.6/dist/animadio.css" rel="stylesheet">

    Then in production, add this one :

    <link href="https://cdn.jsdelivr.net/npm/animadio@0.3.6/dist/min/animadio.min.css" rel="stylesheet">

    But if you only need a part of Animadio, you can use one of those 3 files :

    • animadio-grid for Global & Grid parts
    • animadio-elements for Global & Elements parts
    • animadio-states for Global & States parts
    • Animadio-Grid

      Development :

      <link href="https://cdn.jsdelivr.net/npm/animadio@0.3.6/dist/animadio-grid.css" rel="stylesheet">

      Production :

      <link href="https://cdn.jsdelivr.net/npm/animadio@0.3.6/dist/min/animadio-grid.min.css" rel="stylesheet">
    • Animadio-Elements

      Development :

      <link href="https://cdn.jsdelivr.net/npm/animadio@0.3.6/dist/animadio-elements.css" rel="stylesheet">

      Production :

      <link href="https://cdn.jsdelivr.net/npm/animadio@0.3.6/dist/min/animadio-elements.min.css" rel="stylesheet">
    • Animadio-States

      Development :

      <link href="https://cdn.jsdelivr.net/npm/animadio@0.3.6/dist/animadio-states.css" rel="stylesheet">

      Production :

      <link href="https://cdn.jsdelivr.net/npm/animadio@0.3.6/dist/min/animadio-states.min.css" rel="stylesheet">

  • Package Managers

    For the moment, Animadio is available through NPM, Atmosphere & Packagist ; but it will be available as soon as possible through RubyGems & PyPi !

    • NPM
      $ npm i animadio
      $ yarn add animadio
    • Atmosphere
      $ meteor add animadio:animadio
    • Packagist
      $ composer require animadio/animadio

  • Download

    If you prefer to download the files, you can choose between release or repository.

    • Release
      Get the Latest Release !
    • Repository
      $ git clone https://github.com/animadio/animadio.git
