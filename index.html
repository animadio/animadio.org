CSS Only

Animadio is a CSS framework whose library is composed only of CSS source code: jQuery, or even Javascript, are not included or needed in production ...

Grunt is the only JavaScript app used in the framework developement side for recursive tasks (concat, source map, webkit & min) to build the library file (but it can be used to concat CSS elements of your choice)

You can simply use the library & if you want or if you need : you can overload the CSS variables.

Browsers

Animadio uses PostCSS with autoprefixer to provide a maximum of compatibility for all main browsers, including mainly Firefox, Chrome & Opera, secondarily Edge, Safari & Explorer (for some effects, especially all Skew animations)

Installation

Installation can be done by 3 ways :

  • CDN

    All of those CDN links take automatically the latest version, so you don't need to care about updates !

    During development, add this tag to your HTML: 

    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/animadio@latest/dist/animadio.css">

    Then in production, add one of those tags : 

    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/animadio@latest/dist/min/animadio.min.css">
    <link rel="stylesheet" href="https://unpkg.com/animadio/dist/min/animadio.min.css">

  • Package Managers

    For the moment, Animadio is available through NPM/Yarn, AtmosphereJS/Meteor & Packagist/Composer ; but it will be available as soon as possible through RubyGems & PyPi !

    For NPM/Yarn :

    $ npm i animadio
    $ yarn add animadio

    For AtmosphereJS/Meteor :

    $ meteor add animadio:animadio

    For Packagist/Composer :

    $ composer require animadio/animadio

  • Download

    If you prefer to download the files, you can choose between release or repository.

    For the release, click on the button to choose it (take the latest) :

    Latest Release

    For the repository, just type this in your terminal :

    $ git clone https://github.com/animadio/animadio.git
