Animadio

CSS Framework

How to prepare for takeoff ?

Immediate takeoff with Animadio, the only free & open-source framework that offers real interactivity to the click & much more :

  • Eight types of selectors working with hover, focus & checked states
  • Flexbox & Grid systems included with common options
  • Click anywhere on the page to enable & disable any design effect anywhere else
  • Management of keyframes animations & their options
  • Complete & simple customization with directly modifiable CSS variables

CSS Variables

Animadio is a CSS framework whose library used a lot of CSS Variables...

You can simply use the library or if you want/need : you can overload the CSS variables.

Browsers

Animadio uses PostCSS with autoprefixer to provide a maximum of compatibility for all main browsers, including mainly Firefox, Chrome & Opera, secondarily Edge, Safari & Explorer (except for some effects, especially all Skew animations)

Installation

  • CDN

    During development, add this tag to your HTML :

    <link href="https://cdn.jsdelivr.net/npm/animadio@2.1.1/dist/animadio.css" rel="stylesheet">
    <script src="https://cdn.jsdelivr.net/npm/animadio.js@0.1.11/dist/animadio.js"><script>

    Then in production, add this one :

    <link href="https://cdn.jsdelivr.net/npm/animadio@2.1.1/dist/min/animadio.min.css" rel="stylesheet">

    But if you only need a part of Animadio, you can use one of those files :

    • animadio-box for Global & Box parts
    • animadio-grid for Global & Grid parts
    • animadio-elements for Global & Elements parts
    • animadio-states for Global & States parts
    • animadio-helpers for Global & Helpers parts
    • animadio-events for Global & Events parts

  • Package Manager

    Animadio is available through NPM & Yarn !

    • NPM
      $ npm i animadio
    • Yarn
      $ yarn add animadio

  • Download

    If you prefer to download the files, you can choose between release or repository.

    • Release
      Get the Latest Release !
    • Repository
      $ git clone https://github.com/animadio/animadio.git
