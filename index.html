Animadio

How to prepare for takeoff ?

Immediate takeoff with Animadio, the only free & open-source framework that offers real interactivity to the click & much more :

  • Eight types of selectors working with hundreds of class combinations
  • Possible concatenation of 99% of classes
  • Flexbox & Grid systems included with common options
  • Click anywhere on the page to enable & disable any design effect anywhere else
  • Management of keyframes rules, animations & their options
  • Complete & simple customization with directly modifiable CSS variables
  • More than 99% of pure CSS ; so all of that without JavaScript, it's a joke ?... No ?!

CSS Only

Animadio is a CSS framework whose library is composed only of CSS source code: jQuery, or even Javascript, are not included or needed in production ...

Grunt is the only JavaScript app used in the framework developement side for recursive tasks (concat, source map, webkit & min) to build the library file (but it can be used to concat CSS elements of your choice)

You can simply use the library & if you want or if you need : you can overload the CSS variables.

Browsers

Animadio uses PostCSS with autoprefixer to provide a maximum of compatibility for all main browsers, including mainly Firefox, Chrome & Opera, secondarily Edge, Safari & Explorer (for some effects, especially all Skew animations)

Content

The library distribution is available here :
dist/animadio.css
dist/min/animadio.min.css

But if you need only the Flexbox & Grid system :
dist/animadio-grid.css
dist/min/animadio-grid.min.css

Or if the prepared Elements are useful :
dist/animadio-elements.css
dist/min/animadio-elements.min.css

And the best part, the states :
dist/animadio-states.css
dist/min/animadio-states.min.css

CDN

During development, add one of those tags to your HTML :

<link href="https://cdn.jsdelivr.net/npm/animadio@0.2.24/dist/animadio.css" rel="stylesheet">
<link href="https://unpkg.com/animadio@0.2.24/dist/animadio.css" rel="stylesheet">

Then in production, add one of those tags :

<link href="https://cdn.jsdelivr.net/npm/animadio@0.2.24/dist/min/animadio.min.css" rel="stylesheet">
<link href="https://unpkg.com/animadio@0.2.24/dist/min/animadio.min.css" rel="stylesheet">

But if you only need a part of Animadio, you can use one of those 3 files :

  • animadio-grid who contains only Global & Grid parts
  • animadio-elements who contains only Global & Elements parts
  • animadio-states who contains only Global & States parts

Package Managers

For the moment, Animadio is available through NPM, Atmosphere & Packagist ; but it will be available as soon as possible through RubyGems & PyPi !

  • NPM

    $ npm i animadio
    $ yarn add animadio

  • Atmosphere

    $ meteor add animadio:animadio

  • Packagist

    $ composer require animadio/animadio
